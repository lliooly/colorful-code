# Colorful Code 1.x 冻结规则

Colorful Code 1.x 已进入安全冻结期。`apps/web` 与 `apps/desktop` 只接受安全、稳定性、测试、构建和兼容性修复，不再增加体验功能。

2.0 Server 代码只能进入 `apps/server/src/v2`。`COLORFUL_CODE_V2_ENABLED` 默认关闭；Phase 0A 前不得注册 2.0 路由、后台任务或持久化写入，也不得导入 1.x `SessionStore` 或同时写入新旧存储。

每项例外必须在变更说明中解释风险、兼容性和回滚方法，并通过 Phase -1 测试门禁。
